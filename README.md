# RxMvpAlbum

###### RxJava 实现相册选择

##第一版主要功能：
######1.图片展示：使用王浩的BGA文件读取方式，以线程方式后台一次读取所有的图片
######2.相册单选：项目单选方式个人觉得比较好，避免了大规模的notification。
######3.拍照：拍照可自己设定，通过一个布尔值判断。拍照成功自动跳转到裁剪界面。
######4.图片裁剪：裁剪目前只可裁剪成正方形，暂时还未优化。
######5.Rxbus:项目中使用收集的RxBus，简单需求基本可以满足。

##第二版主要功能：
######1.相册多选
######2.可简单设置相册模式
